# NeuroAdapt-AI
NeuroAdapt AI is a cognitive-state-aware GenAI learning system that adapts teaching strategies in real time. It analyzes user behavior like response time and errors to detect confusion or engagement, then generates personalized explanations using dynamic AI-driven pedagogy for better understanding and retention.
